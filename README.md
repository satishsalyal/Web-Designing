## 🎯 Course Objective

The main objective of this course is to provide students with practical experience in designing, developing, managing and optimizing WordPress websites. It covers theme customization, content management, plugins and SEO.

---

## 📚 Course Units

### Unit I — HTML Fundamentals

What is HTML. HTML Documents, Basic structure of an HTML document, Creating an HTML document. Mark up Tags. Heading, Paragraphs, Line Breaks. HTML Tags, Introduction to elements of HTML, Working with Text. Working with Lists, Tables and Frames. Working with Hyperlinks, Images and Multimedia, Working with Forms and controls.

---

### Unit II — Cascading Style Sheets (CSS)

Concept of CSS, Creating Style Sheet, CSS Properties, CSS Styling (Background, Text Format, Controlling Fonts). Working with block elements and objects. Working with Lists and Tables.

---

### Unit III — Introduction to WordPress

**Overview & Setup**
- Overview of WordPress as a CMS, Features and advantages
- Installing WordPress on Windows
- Installing WordPress on Local Web Server

**WordPress Posts**
- Adding Posts, Editing Posts, Deleting Posts, Previewing Posts, Publishing Posts
- Adding Links, Editing Links, Deleting Links
- WordPress Media Management

**WordPress Pages**
- Adding Pages, Editing Pages, Previewing Pages, Deleting Pages, Publishing Pages

---

### Unit IV — WordPress Management & Customization

**WordPress Comments**
- Adding Comments, Editing Comments

**WordPress Categories**
- Adding Categories, Editing Categories, Deleting Categories, Arranging Categories

**WordPress Tags**
- Adding Tags, Editing Tags, Deleting Tags

**WordPress Appearance Customization**
- Theme Management, Customizing Themes
- Widget Management, Managing Menus, Changing Backgrounds

**WordPress Plugins**
- Plugin Settings, Installing Plugins

## 📝 Note for Paper Setting

The External End Semester Theory Examination will have total marks **60**. The question paper will be divided into two sections — **Section A** & **Section B**.

- **Section A** — 04 compulsory short answer type questions, one from each unit, each carrying **03 marks**.
- **Section B** — Eight (08) long answer questions, two from each unit. The student will have to attempt one (01) question from each unit. Each question will carry **12 marks**.
